Physiology Data Definitions
===========================

We recommend storing each physiological modality in a separate, modality-specific file.

Column naming conventions for ``_physio.tsv`` files should follow the rules defined in the respective modality-specific pages. This applies both when data are stored in separate files per modality and when multiple modalities are combined into a single set of data files.

Guidelines for individual modalities are given below.
